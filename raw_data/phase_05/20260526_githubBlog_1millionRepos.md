## TLDR
GitHub hit 1 million repositories on July 24–25, 2010 — approximately 27 months after public launch in April 2008. The blog post celebrated the milestone by highlighting notable projects hosted: jQuery, Ruby on Rails, Node.js, Redis. No user count was given in the post itself. The post ended with "Onward to ten million!" — GitHub doubled to 2M repos by 2011. The first million took 3 years, 8 months, 15 days; subsequent millions came faster as network effects compounded. GitHub reached 100K users within first year (by mid-2009), confirming explosive early word-of-mouth growth.
