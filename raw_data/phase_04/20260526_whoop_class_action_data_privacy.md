## Key Facts
- Lomeli v. Whoop, Inc. — Northern District of California, filed Aug 2025
- Allegation: Whoop embedded Segment (third-party tracker) in mobile app without consent
- Segment collected sensitive personal data and transmitted to third parties without user knowledge
- Whoop privacy policy: permits sharing "aggregated and de-identified" data for "research and commercial purposes"
- 2024 research (Imperial College London): de-identified fitness data can be re-identified with 87% accuracy using 3 data points: age range, zip code, activity pattern
- Washington My Health My Data Act (eff. March 2024): requires opt-in consent for consumer health data

## Broader Context
- Whoop valuation: $3.6–3.7B (2025); $260M ARR; not publicly disclosed user count
- Users in 56 countries; three tiers: $199/$239/$359/year

## Relevance
Key warning about fitness/health data B2B monetization risks; covert tracking = class action trigger
