# Hybrid Mode Analysis — Four Archetypes

## 1. Strava Mode: Private Input + Public Feed, Opt-In

### How It Works
- Default: all activities public (new accounts default to "Everyone")
- Private layer: heart rate, health metrics, medical notes — private by default, user-controlled
- B2B: aggregate public route/activity data → Strava Metro → city planners, researchers
- Social: activity feed, segments, kudos, clubs — all default public

### Real Data
- 180M users (2025); $415M+ revenue; 90% from premium subscriptions
- Strava Metro: 3,500+ qualified org users; offered free to governments (goodwill/PR)
- Privacy: first/last 200m of routes hidden by default (added after 2018 military base controversy)
- Default = public confirmed: support.strava.com official privacy defaults page

### What Works
- Default public creates massive social graph and content flywheel without requiring users to actively share
- Users do NOT feel exposed because the public data (activity route) feels low-stakes
- High-sensitivity data (HR, health) kept strictly private → maintains trust
- B2B data product uses only the public/aggregate layer

### Our Fit
- HIGH: talent platform where public work signals (projects, endorsements, activity) are low-sensitivity; private compensation, health, or personal goals stay private
- Default public + clearly private layer = Strava playbook

## 2. LinkedIn Mode: Default Public, Salary Private

### How It Works
- Profile = public by default (name, photo, headline, experience)
- Salary data = private by default (only surfaces as aggregated "Salary Insights")
- B2B: public profiles → Recruiter product → employers pay $8K–$100K+/yr
- Revenue: Talent Solutions = 60%+ of $17.1B (2024)

### Real Data
- 1.3B members; 310M MAU; 134.5M DAU; DAU/MAU = 43.4%
- Q4 2011 Hiring Solutions: $84.9M = 50% of revenue (SEC 8-K)
- 2024: LinkedIn revenue $17.1B; Talent Solutions ~$10B
- Premium crossed $2B/12mo (Jan 2025, TechCrunch)

### What Works
- Users get free platform in exchange for public profile = implicit value trade
- Salary privacy protects users from discrimination while aggregate data still flows to B2B product
- Network effect: your connections' profiles make yours more valuable to Recruiter

### Our Fit
- HIGH: same model — user creates public professional identity; platform sells aggregated intelligence to employers
- Risk: 2024 LinkedIn AI training backlash shows users dislike covert data use; transparency required

## 3. GitHub Mode: Public Default, Private = Paid

### How It Works
- Before 2019: private repos required paid plan (~$7/mo)
- Post-2019: private repos free for individuals; private org repos require Teams ($4/user/mo)
- Copilot: trained on public code → sold as AI coding assistant at $10–$39/user/mo

### Real Data
- 630M repos; 395M public (63%); ~235M private (37%)
- Copilot: 4.7M paid subs (Jan 2026); 20M total users; 90% Fortune 100 adoption
- Revenue: $2B ARR total; Copilot = "larger business than all of GitHub at acquisition" (Nadella)
- Microsoft acquired for $7.5B (2018)

### What Works
- Open-source ethos: developers WANT their code public (reputation, portfolio)
- The public default is culturally appropriate and welcomed, not imposed
- Microsoft monetized the public code corpus via Copilot without users feeling violated (mostly)
- Private = paid creates a natural upgrade path for enterprises

### Our Fit
- VERY HIGH: if our users are AI practitioners/researchers, public project work = GitHub repos, papers, models
- Private = paid feature can apply: private AI projects, confidential experiments
- Risk: GitHub Copilot faced some developer backlash about code scraping; IP issues ongoing

## 4. Notion Mode: Private Default, Public = Escape Valve

### How It Works
- Everything private by default; users manually publish individual pages
- No social feed; no discovery mechanism
- Public pages = Notion Sites (static publishing); no social engagement layer
- Monetization: pure SaaS subscription; no B2B data product

### Real Data
- 100M users; 4M paying; $400M ARR (2024); $500–600M ARR (2025)
- ARPU ≈ $100/paying user/year (inferred)
- Revenue growth: $67M (2022) → $250M (2023) → $400M (2024)

### What Works
- Deep workspace lock-in: switching costs very high after 100+ pages/databases
- AI upsell works because users trust their data is private
- Public publishing = escape valve, not core experience

### Our Fit
- LOW: private default doesn't create the public signals needed for B2B talent intelligence
- Works for productivity SaaS; does not work for our specific B2B model

| GitHub (public default + private paid) | VERY HIGH | HIGH | HIGH | Applicable if cultural fit |

## Recommendation (data-only, no analysis judgment)
The data suggests Strava Mode + LinkedIn Mode hybrid: default public for professional work signals (projects, skills, endorsements) + explicitly private for sensitive personal data (compensation, health, confidential projects) + aggregate public signals → B2B employer/recruiter product.
