# X (Twitter) — Default Public Microblogging

## Source date range: 2024–2025

## Scale
- ~600M monthly active users (MAU) claimed by Elon Musk (2023); independently unverified
- ~250M daily active users (DAU) — company does not report DAU officially since 2022
- Revenue: declined from ~$5.1B (2021) to ~$2.5B (2023) post-acquisition restructuring
- Musk acquisition (Oct 2022): $44B; X Corp now private, no mandatory SEC reporting

## Default Privacy Setting
- Default = public: all posts visible to anyone, logged-in or not
- Private (Protected) tweets: manual opt-in; follower requests required
- No official stat on % users with protected accounts; estimated 5–10% based on research

## % Users Public
- ~90–95% of accounts estimated public (default + most never change)
- Protected accounts: small minority
- Verified/Blue accounts: all public by design

## D30 / D90 Retention
- Not publicly disclosed post-Musk
- Historical Twitter 6-month retention (Lenny Rachitsky framework): "good" = 25% for consumer social

## Monetization Path
- Advertising: primary, but declining
- X Premium (Blue): $8–$16/month (verification + features)
- Grok AI: integrated AI assistant for Premium
- Data API: X charges $42K/month for full data access (raised 2023)
- xAI: separate company training on X data

## Design Pattern
- "Threads" within X = semi-private long-form
- Communities = interest-based private groups within public app
- The public default + reply graph = network effect flywheel

## Fit with "Feed B-Side Data" Goal
- LOW fit: chaotic data quality, user trust damaged, not professional
