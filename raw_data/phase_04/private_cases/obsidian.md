# Obsidian — Local-First, Extreme Privacy

## Source date range: 2023–2025

## Growth Curve
- Launched 2020; bootstrapped, 18 employees (2026)
- Estimated ~1M–1.5M users (2023 Fast Company estimate based on GitHub download counts)
- CEO Steph Ango LinkedIn post (2025): "No one knows how many users Obsidian has" — company deliberately does not track
- Discord: 110,000+ members; Reddit: ~94,600 members (top 5% of Reddit communities)
- Revenue: $2M ARR (Latka estimate 2025) — likely an undercount; primary revenue = Obsidian Sync + Publish subscriptions
- No VC funding; no external investors

## D30 / D90 Retention
- Unknown; company does not disclose
- Plugin ecosystem data: plugin version install counts provide proxy for active users
- Forum thread (2024): community estimated active vault users ~500K–800K based on plugin telemetry

## Monetization Path
- Core app: free forever (local)
- Obsidian Sync: $4/mo (end-to-end encrypted cloud sync)
- Obsidian Publish: $8/mo (public website from vault)
- Catalyst one-time: $25–$100 (early supporter)
- Commercial license: $50/user/year for businesses
- No ads, no investor pressure, intentionally small

## Data Realness
- True local-first: data never leaves device unless user enables Sync
- Obsidian Sync uses E2E encryption; Obsidian cannot read content
- No telemetry by default (users can opt in to analytics)

## % Users Public
- Obsidian Publish users: very small fraction
- Estimated <1% publish public vaults
- Default = 100% private

## Fit with "Feed B-Side Data" Goal
- ZERO fit: architecture makes B2B data impossible
- Lesson: extreme privacy = passionate but small, high-ARPU niche; not scalable to B2B data play
