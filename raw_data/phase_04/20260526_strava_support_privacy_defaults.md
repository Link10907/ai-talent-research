# TLDR: Strava Support — Privacy Defaults for New Accounts

**Source:** support.strava.com/hc/en-us/articles/360034758331-Your-Privacy-Defaults-When-You-Create-a-Strava-Account  
**Date:** Updated March 2026 (original date varies)  
**Type:** company_blog (official support documentation)

## Key Facts
- New Strava accounts default: Profile = Everyone; Activities = Everyone; Maps = Everyone
- First activity uploaded → future activities auto-hide first/last 200m (privacy zone)
- Users can change settings to Followers Only or Only You at any time
- Privacy zones configurable around home/work addresses

## On the "70% Public" Claim
- This specific statistic is NOT in any official Strava document found
- The claim is inferrable from: default = public + most users never change defaults → majority public
- Should be cited as "default public confirmed; 70% figure unverified, treat as inferred"
