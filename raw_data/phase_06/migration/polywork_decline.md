# Polywork — Decline & Shutdown

## Timeline
- **Founded**: 2021; raised $13M (backers: Collison brothers/Stripe, Alexis Ohanian, Elad Gil)
- **Peak**: 2021–2022; positioned as "LinkedIn alternative" for multi-hyphenate professionals
- **Pivot**: End of 2023 — shifted from professional network to website creation tool
- **Shutdown date**: January 31, 2025
- **Decision**: Founder Peter Johnston announced shutdown; returned remaining capital to investors

## User Data
- **Peak user count**: Not publicly disclosed
- **Post-pivot retention**: Failed — could not gain traction in website builder market
- **Funding exhausted**: Shutdown triggered by no viable path to "gigantic business"

## Why It Failed
1. User growth stalled after initial hype
2. Network effect never materialized — couldn't reach LinkedIn critical mass
3. Pivot to website builder competed with Webflow, Linktree, Carrd — equally commoditized
4. Core value prop ("show multi-dimensional self") had weak job-market feedback loop

## Migration After Shutdown
- No dominant successor; users dispersed
- LinkedIn remained default despite user frustration
- Some moved to personal sites (Carrd, Webflow)
- Read.cv was also shutting simultaneously — no alternative professional network

## Key Signal
- "Better LinkedIn" is a crowded, hard-to-win position
- Professional networks require job market integration to have stickiness
- Aesthetic differentiation alone insufficient for migration + retention

## Evidence Quality
- Shutdown: **High** (founder tweet, widely covered)
- User count: **Unknown** (never disclosed)
- Reasons: **Medium** (founder statement + analyst commentary)
