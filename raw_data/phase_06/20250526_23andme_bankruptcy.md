# TLDR: 23andMe — Data Exchange Willingness & Trust Collapse

**Date collected**: 2025-05-26
**Source type**: NPR, NEJM, Harvard Gazette, Lawfare, Drug Discovery Trends

## Key Numbers
- Users at peak: 15M+ genetic profiles
- Research opt-in rate: 80% of users
- GSK deal (2018): $300M investment for 5M customer data access
- GSK extension (2022): $50M additional payment
- Bankruptcy filed: March 23, 2025
- TTAM acquisition price: $305M for DNA database
- State AGs suing to block: 28

## The 80% Opt-In Signal
80% of users agreed to have de-identified DNA used for pharma drug discovery. This is extraordinary — most users think of it as "helping science" not "selling data." The consent framing matters enormously.

## The Trust Collapse Pattern
Company peak → hidden secondary use (GSK deal awareness) → stock collapse → data breach (2023, 7M records) → bankruptcy → data sale = total trust destruction. The trust capital built over years evaporated in months.

## Key Lesson
Users will donate intimate data (DNA) for personalized insight + science contribution framing. But data is not infinitely transferable — it belongs to the original consent context. Bankruptcy = consent context destroyed.

## Confidence
High for all figures. Extensively reported across multiple credible outlets.
