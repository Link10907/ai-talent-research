# Strava — User Data Sharing Willingness

## Platform Scale (2024–2025)
- **Registered users**: 135M (2024), 180M (2025)
- **Countries**: 190+
- **Revenue**: $163.4M (2024), $415M (2025)
- **Paid subscribers**: ~2% of registered users (low penetration for freemium fitness)
- **Revenue model**: 90% from premium subscriptions; 10% from sponsored challenges + aggregated data sales

## Default Privacy Settings
- Activities and profiles default to **"Everyone"** (public) unless changed
- Users must opt-out of public visibility (opt-out not opt-in model)
- Even "Followers-only" accounts maintain a minimal public profile page
- Global Heatmap: aggregate GPS data from all activities shown publicly — users can opt out

## Data Sharing Behavior
- Users willingly share:
  - GPS routes (location)
  - Heart rate data
  - Speed/power/cadence metrics
  - Social connections (followers/following)
- Exchange: "Compare with friends" and "Segment leaderboards" — social competition is primary motivator
- Strava Metro: sells anonymized route data to city governments for urban planning (opt-out available)

## Privacy Incidents
- **2018**: Strava Global Heatmap revealed locations of secret military bases (CIA, NSA, etc.) — massive PR incident
- Response: Added opt-out for military personnel; no structural change for civilian users
- **2024 Privacy Audit**: Grade F (32/100) from terms.law — collects extensive data, shares with third parties, retention policy unclear

## Willingness-to-Share Pattern
- Social comparison is the core motivator for data sharing on Strava
- Users accept GPS/biometric data collection as inherent to the product
- Low awareness of secondary data uses (Strava Metro, research partnerships)
- Backlash when hidden (military base incident) >> backlash when disclosed

## Key Insight
Users share location + biometric data when:
1. The value exchange is immediate and visible (leaderboards, route maps, friend comparison)
2. They can contextualize it within a social/athletic identity
3. Secondary uses are abstract and distant

## Evidence Quality
- User counts: **High** (Business of Apps, Sacra)
- Revenue: **High** (multiple sources cross-referenced)
- Privacy incident: **High** (widely documented in 2018)
- Data sharing behavior: **Medium** (documented but not user-survey-based)
